# Platform Crawling Server
파이썬, 장고를 이용하여 유튜브,트위치,아프리카의 셀럽들의 정보를 크롤링하여 데이터를 가공하여 DB에 저장합니다. 

# back-end 사용 스택
Python, Django, selenium
